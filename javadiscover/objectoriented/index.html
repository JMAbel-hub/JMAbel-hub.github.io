Object Oriented Programming

Learn about object creation in Java and how to apply them.

Classes and Objects:

Classes and objects are the two main aspects of object-oriented programming. The whole concept of object-oriented programming revolves around creating objects of a certain class. For example if you had a class named Fruit you would have objects such as Apple, Banana, Mango which are created off of the blueprint of Fruit. To do this an entirely new class dedicated to making objects with the object having unique features such as size, name, age that make each object completely unique.
