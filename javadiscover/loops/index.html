Introduction to Looping and Conditions

Learn about loops and conditions in Java and how to use them.

If Statements:

The if statement is used to gate blocks of code off unless a condition is met. If a condition is met then the code in the body gets executed otherwise it is skipped over. These can be chained together through else and else if statements in order to test multiple condition cases.

Loops:

In programming looping is considered a fundamental that facilitates of set instructions or functions repeatedly until you reach a desired goal or until a condition evaluates to true. There are several instances of looping in Java with the main three to consider being: for, while, and do-while with each having their unique purpose and some being better under certain conditions.

While Loop:
The while loop executes code based upon a given boolean condition and continues to run and loop until the conditions are no longer met. While the condition is true, the loop body executes until the condition becomes false where the loop terminates and ends its life cycle.

Do-While Loop:
The do-while loop is extremely similar to the while loop with some modifications to make it unique. Specifically, it checks for condition after executing a loop, meaning that everything in the loop body will be executed for it checks the boolean condition.

For Loop:
The for loop provides a a very concise way of writing a loop structure. Unlike the while loops, the for loop deals with the conditions and initialization and incrementation/decrementation all in one line. This being said they are similar in that as soon as the condition is no longer met, then the loop terminates.
