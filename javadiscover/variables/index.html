Variables

Learn about datatypes in Java and how to manipulate them.

Primitive Data Types:

Java's most basic data types are referred to as primitive data types and are included within the system by default. A lot of the data types deal with handling numbers with a couple exceptions in char, boolean, and null. For storing numbers, the most frequently used are int and double but feel free to experiment with the other data types.

  • int: holds a positive or negative non-decimal number
  • char: holds a single character
  • boolean: holds a true or false value
  • byte: holds a 8-bit signed two's complement integer
  • long: holds a 64-bit signed two's complement integer
  • short: holds a 16-bit signed two's complement integer
  • double: holds a 64-bit representation of a floating-point number
  • float: holds a single-precision value that has a width of 32 bits in storage
  • null: holds a null value meaning nothing

Math Operations and Comparsions:

Basic math operations can only be applied to ints, doubles, and floats with it not being supported by the other data types. The basic math operations are as follows:

  • + addition: returns the sum of two elements
  • - subtraction: returns the difference two elements
  • * multiplication: returns the product of two elements
  • / division: returns the quotient of two elements
  • % modulo: returns the remainder when dividing

Comparison operators can be used to compare two values and return a boolean value of true or false. These vary in purpose, however, they are extremely useful for things such as sorting which you will eventually learn.

  • > greater than: returns whether the left value is greater than right
  • < less than: returns whether the left value is less than the right
  • >= greater than or equal to: returns whether the left value is greater than or equal to the right
  • <= lesser than or equal to: returns whether the left value is lesser than or equal to the right
  • == equal to: returns whether two values are equal to one another
  • != not equal to: returns whether two values are not equal to one another

Strings:

Another very important thing to remember is the string data type even though it isn't considered to be a primitive data type. They can be created by putting characters between a pair of double quotes (") and can be compared by using the equals() method which is comparable to the == learned before and returns a boolean which is either true or false depending on whether the two strings match.

Activity:

For the activity in stage 2, create a new class named stageTwo and just mess around with the operators and comparisons in order to get comfortable with them. These are simply the building blocks for us to do more fun stuff later on so why not get used to using them now!
